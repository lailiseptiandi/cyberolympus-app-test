# TATA CATA MENJALANKAN

1. Rename .env.example to .env
2. Buka file .env dan ubah

DB_DATABASE=nama_database

DB_USERNAME=root

DB_PASSWORD=password (jika ada)

3. JALANKAN,

composer install

php artisan migrate

Jalankan Seeder untuk generate akun & data dummy customer :

php artisan db:seed

Terakhir Jalankan :

php artisan serve
